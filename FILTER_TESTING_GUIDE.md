# CutieCam Filter Testing Guide

## フィルターテストガイド

このガイドでは、CutieCamの新しいフィルターシステムのテスト方法を説明します。

---

## 📸 テストの準備

### 推奨テスト画像
1. **ポートレート写真** - 人物の肌の質感をチェック
2. **風景写真** - 色のグラデーションと全体的な雰囲気をチェック
3. **明るい環境** - ハイライトとハレーション効果をチェック
4. **暗い環境** - シャドウとフェード効果をチェック
5. **カラフルな被写体** - 彩度と色温度の調整をチェック

---

## 🎨 各フィルターの期待される結果

### 1. Natural（ナチュラル）
- **説明**: フィルターなし、オリジナルの画像
- **用途**: フィルターなしで撮影したい場合、または調整バーだけを使いたい場合
- **期待される結果**: 画像に変化なし
- **調整可能**: すべてのパラメータ

### 2. Film Dreams（フィルムドリーム）
- **説明**: 温かみのあるヴィンテージフィルムルック
- **特徴**:
  - 温かい色調（オレンジ・黄色寄り）
  - 適度なフィルムグレイン
  - 軽いビネット効果
  - 柔らかいハイライト
- **期待される結果**: ノスタルジックで柔らかい雰囲気
- **適した被写体**: ポートレート、風景、夕暮れ

### 3. K-Pop Glow（ケーポップグロー）
- **説明**: 柔らかく発光する肌のような K-pop スタイル
- **特徴**:
  - 明るめの露出
  - ハレーション効果（発光感）
  - 柔らかいピンク・ピーチ系の色調
  - 最小限のグレイン
- **期待される結果**: 肌が滑らかで発光しているような仕上がり
- **適した被写体**: ポートレート、セルフィー

### 4. Disposable（ディスポーザブル）
- **説明**: Y2K 使い捨てカメラの雰囲気
- **特徴**:
  - 鮮やかな色彩（高彩度）
  - オレンジ系のライトリーク
  - 目立つフィルムグレイン
  - 強めのビネット
- **期待される結果**: 2000年代の使い捨てカメラで撮ったような仕上がり
- **適した被写体**: パーティー、イベント、友達との写真

### 5. Y2K Digital（ワイツーケーデジタル）
- **説明**: 2000年代初期のデジタルカメラ美学
- **特徴**:
  - デジタルノイズ（カラフルなノイズ）
  - やや鮮やかな色彩
  - 最小限のグレイン
  - クールな色調
- **期待される結果**: 初期のデジタルカメラで撮影したような質感
- **適した被写体**: ストリート、都市、ノスタルジックな写真

### 6. Soft Aesthetic（ソフトエスセティック）
- **説明**: Instagram 向けの柔らかい夢のようなルック
- **特徴**:
  - 高いフェード効果（黒が持ち上がる）
  - 強めのハレーション
  - ピンク系の色調
  - 柔らかいコントラスト
- **期待される結果**: 夢のような、エセリアルな雰囲気
- **適した被写体**: ファッション、美容、ライフスタイル

### 7. Cinematic（シネマティック）
- **説明**: 映画のようなカラーグレーディング
- **特徴**:
  - 青みがかった色調
  - 強いコントラスト
  - 抑えたハイライト、持ち上げたシャドウ
  - 適度なビネット
- **期待される結果**: 映画のようなドラマチックな仕上がり
- **適した被写体**: ドラマチックなシーン、ポートレート、風景
- **注**: プレミアム機能（実装時）

### 8. B&W Film（白黒フィルム）
- **説明**: クラシックな白黒フィルム
- **特徴**:
  - 完全なモノクロ
  - フィルムグレイン
  - 適度なコントラスト
  - ビネット効果
- **期待される結果**: クラシックな白黒フィルム写真の質感
- **適した被写体**: ポートレート、建築、ストリートフォト

## 🎛️ 調整バーのテスト

### BASIC（基本調整）

#### 1. 露出（Exposure）
- **範囲**: -1.0 〜 +1.0
- **テスト方法**: 
  - -1.0 に設定 → 画像が暗くなる
  - +1.0 に設定 → 画像が明るくなる
  - 0.0 に戻す → 元の明るさに戻る
- **期待される動作**: スムーズに明暗が変化

#### 2. コントラスト（Contrast）
- **範囲**: 0.5 〜 1.5
- **テスト方法**:
  - 0.5 に設定 → 平坦な画像（明暗差が少ない）
  - 1.5 に設定 → はっきりした画像（明暗差が大きい）
  - 1.0 に戻す → デフォルトのコントラスト
- **期待される動作**: 明暗の差が変化する

#### 3. 彩度（Saturation）
- **範囲**: 0.0 〜 1.5
- **テスト方法**:
  - 0.0 に設定 → モノクロに近い
  - 1.5 に設定 → 非常に鮮やかな色
  - 1.0 に戻す → 標準の彩度
- **期待される動作**: 色の鮮やかさが変化する

### FILM EFFECTS（フィルム効果）

#### 4. フィルムグレイン（Grain）
- **範囲**: 0.0 〜 0.8
- **テスト方法**:
  - 0.0 に設定 → グレインなし
  - 0.8 に設定 → 目立つフィルムグレイン
- **期待される動作**: ランダムなテクスチャが追加される

#### 5. ビネット（Vignette）
- **範囲**: 0.0 〜 0.8
- **テスト方法**:
  - 0.0 に設定 → ビネットなし
  - 0.8 に設定 → 四隅が暗くなる
- **期待される動作**: 画像の端が徐々に暗くなる

#### 6. フェード（Fade）
- **範囲**: 0.0 〜 0.5
- **テスト方法**:
  - 0.0 に設定 → フェードなし
  - 0.5 に設定 → 黒が持ち上がり、色褪せた雰囲気
- **期待される動作**: 古い写真のような色褪せた感じ

#### 7. ハレーション（Halation）
- **範囲**: 0.0 〜 0.6
- **テスト方法**:
  - 0.0 に設定 → ハレーションなし
  - 0.6 に設定 → 明るい部分が発光する
- **期待される動作**: ハイライト部分が柔らかく光る

### ADVANCED（高度な調整）

#### 8. ハイライト（Highlights）
- **範囲**: -0.5 〜 +0.5
- **テスト方法**:
  - -0.5 に設定 → 明るい部分が暗くなる
  - +0.5 に設定 → 明るい部分がさらに明るくなる
  - 0.0 に戻す → デフォルト
- **期待される動作**: 画像の明るい部分のみが変化

#### 9. シャドウ（Shadows）
- **範囲**: -0.5 〜 +0.5
- **テスト方法**:
  - -0.5 に設定 → 暗い部分がさらに暗くなる
  - +0.5 に設定 → 暗い部分が明るくなる
  - 0.0 に戻す → デフォルト
- **期待される動作**: 画像の暗い部分のみが変化

#### 10. 色温度（Temperature）
- **範囲**: -0.5 〜 +0.5
- **テスト方法**:
  - -0.5 に設定 → 寒色系（青み）
  - +0.5 に設定 → 暖色系（オレンジみ）
  - 0.0 に戻す → ニュートラル
- **期待される動作**: 画像全体の色温度が変化

## 🐛 よくある問題と解決方法

### 問題1: フィルターが適用されない
- **原因**: 画像の変換エラー
- **解決方法**: 別の画像で試す、アプリを再起動

### 問題2: 調整バーが反応しない
- **原因**: 処理中にバーを操作した
- **解決方法**: 前の処理が完了するまで待つ

### 問題3: 画像がおかしくなる
- **原因**: 極端なパラメータ値
- **解決方法**: Reset ボタンで初期値に戻す

### 問題4: 処理が遅い
- **原因**: デバイスの性能、画像サイズ
- **解決方法**: 
  - シミュレーターではなく実機でテスト
  - Release ビルドでテスト
